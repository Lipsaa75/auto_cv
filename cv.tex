\section{Project}

\begin{jobshort}{Exhaled Ammonia Monitoring System (EAMS)}{Jan 2026 - present}
The Breathing to Kidney Disease Detection System is a biomedical project that uses human breath analysis to detect possible kidney dysfunction non-invasively. Gas sensors detect ammonia and urea-related compounds in breath samples, providing an early warning mechanism for kidney-related disorders.
\end{jobshort}


\begin{joblong}{ALC-LOCK Protocol (Alcohol-Lock)}{Apr 2026 - present}
The Alcohol Sensing Alert with Engine Locking System is a safety-based embedded system designed to sense the alcohol level in a driver’s breath and stop vehicle operation when the alcohol level exceeds a predetermined threshold. The system helps reduce accidents caused by drunk driving by automatically disabling the ignition system.
\end{joblong}

\begin{joblong}{Smart Pill Box}{Oct 2025 - Jan 2026}
The Smart Pill Box is an IoT-driven healthcare system bridging hardware and software. Utilizing a sensor-equipped box and a Flutter app synced via Firebase, it tracks real-time medication adherence. This closed-loop system triggers alerts and logs intake, demonstrating strong integration of Embedded Systems and mobile architecture to improve patient outcomes.
\end{joblong}
